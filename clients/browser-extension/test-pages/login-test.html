AMPass Extension Test Page

Use this page to test form detection, autofill, autosave, and password update prompts.

How to test: 1. Load the AMPass extension (unpacked) in Chrome/Edge
2. Connect to your AMPass server and unlock the vault
3. Add a vault item with URL "localhost" or this page's URL
4. Reload this page — the extension should detect the login form
5. Click the AMPass icon on the password field or use the popup to autofill
6. Submit the form to test autosave prompt
7. Change the password and submit again to test update prompt
