# AMPass PHP Web App — Release Package

## Contents

Copy the following from `ampass/` to create the release package:

```
ampass/
├── index.php
├── .htaccess
├── sw.js
├── manifest.webmanifest
├── config/
│   ├── .htaccess
│   └── config.sample.php          (NOT config.php — user generates this)
├── database/
│   ├── .htaccess
│   ├── schema.sql
│   └── migrations/
│       └── 001_extension_tables.sql
├── install/
│   ├── .htaccess
│   └── index.php
├── docs/
│   ├── .htaccess
│   └── extension-api.md
├── app/
│   ├── .htaccess
│   ├── core/
│   ├── controllers/
│   ├── models/
│   └── views/
└── public/
    ├── css/app.css
    ├── js/*.js
    └── assets/
```

## Excluded from release

- `config/config.php` (contains secrets — generated by installer)
- `config/.install_lock` (created by installer)
- Any `.env` files
- Any test databases
- Any private keys
- `node_modules/`
- `.git/`
